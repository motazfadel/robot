# Joe Mobile

هذه أصبحت نواة موبايل عملية لـ `جو` بدل مجرد دردشة تجريبية.

## ما الموجود الآن

- واجهة قيادة يومية موجهة لسيناريو علاء
- قاعدة بيانات محلية `Room`
- ملخص حي للتذكيرات والديون والفواتير والمشتريات
- أوامر عربية محلية مثل:
  - `شو عندي اليوم`
  - `ملخص اليوم`
  - `سجل دين على أبو رامي 300 دولار بعد شهر`
  - `أضف سكر إلى المشتريات`
  - `سجل تذكير زيارة الطبيب غدا`
- إدخال نصي وصوتي
- رد صوتي عبر `TextToSpeech`
- بيانات أولية seeded لتجربة الواجهة فور التشغيل

## الاتجاه المعماري

النسخة الحالية مبنية لتكون الأساس الصحيح لـ:

- `Hotword` محلي لاحقًا
- تخزين مشفر لاحقًا
- مزامنة عائلية لاحقًا
- OCR وفواتير ذكية لاحقًا
- ربط `OpenAI` للمهام الثقيلة لاحقًا

## الوضع الحالي

هذه ليست بعد نسخة النداء الدائم `جو` في الخلفية، لكنها أصبحت قاعدة حقيقية لتطبيق إداري ذكي على أندرويد، وليست مجرد نموذج محادثة.
